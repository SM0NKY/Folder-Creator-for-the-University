Este es un proyecto para organizar archivos para la Universidad, permite crear de forma autonoma carpetas por c/semana asi 
ahorrar tiempo en el proceso, sin la necesidad de contar las semanas ni perder la cuenta.

This is a project to organize files for the University, it allows you to create autonomously folders for each week, this way
you save time in the process, without the need to count the weeks and without losing the count.
